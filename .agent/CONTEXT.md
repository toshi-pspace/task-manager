# Context — Task Manager

## What this is
A simple to-do list app. The user's first project to learn Claude Code + the PARA system.

## Stack & tools
- Plain HTML + CSS + vanilla JavaScript. No framework, no build, no install.
- Storage: browser localStorage (key `task-manager.tasks`).

## Conventions
- All app code lives in `src/` (index.html, style.css, app.js).
- Keep it dependency-free unless there's a real reason to add tooling.

## Key commands
```
# No build needed. To run, just open the file:
open "src/index.html"
```
