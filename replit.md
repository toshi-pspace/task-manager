# Agent Instructions (Replit)

This project uses a plain-text memory system shared across all tools.

1. **Read `.agent/STATE.md` first** — that's where the last session left off.
2. Reference `.agent/CONTEXT.md` (stack, conventions, commands) and `.agent/refs/` (files).
3. Log significant choices in `.agent/DECISIONS.md`.
4. **Before finishing, update `.agent/STATE.md`** (Now / Next / Open questions).

Sync: this Repl is connected to GitHub. Pull before starting, push when done so
local tools (Claude Code, Codex) see the same memory. See `docs/guides/replit-sync.md`.
Durable knowledge belongs in `docs/`. Keep `STATE.md` lean and current.
